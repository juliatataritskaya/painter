Keyboard shortcuts

  • 'P' - Pen tool
  • 'V' - Vertical mirror pen tool
  • 'B' - Paint bucket tool
  • 'E' - Eraser tool
  • 'X' - Swap primary/secondary colors
  • 'N' - Create new empty frame
  • 'R' - Remove frame
  • 'D' - Dublicate frame
  • 'T' - Rotate canvas tool
  • 'M' - Mirror canvas tool
  • 'K' - Open an opened popup
  • 'ESC' - Close an opened popup
